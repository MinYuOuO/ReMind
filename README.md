# ReMind

BAI13123 Mobile Application Development Project

## Project Overview

**ReMind** is a privacy-first mobile relationship manager built with  **Ionic + Angular + Firebase**.

It helps users record and maintain friendships through structured profiles, notes, and offline-first caching.  

This repository contains the full source code of *ReMind Application.*


## Quick Start Guide

   ```

   cd ReMind
   ```
